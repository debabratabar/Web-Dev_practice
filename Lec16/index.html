Display Property

Display property defines how a element should appear in the Webpage



1. Inline Element
inline elements tries to fit same line Inline elements covers the context size Inline element doesn't accept customized height & width value Inline Element doesn't accept vertical margin


2. Block Element
  • Block Elements start with New line
  • BLock Elements starts from left and go all the way to right of the viewport
  • BLock Elements supports customize height & width value
  • Block Elements supports customize padding & margin
