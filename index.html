Learning of CSS

CSS stands for Cascading Style Sheet , It handles the styling of a Webpage ,

Different types of CSS

Importance of Cascading word in CSS

Cascading is roled based mechanism to priority or hierarchy to applying styles to your html elements . its defining how the styles will be added to html elements when we apply multiple style to a sigle html element

Different Four Factors of Cascading

  1. Specificity

    2. Specificity follows below rule
    (0/1) --> Inline
    (0/1) --> ID
    (0/1) --> class
    (0/1) --> Element

  2. Importance

    3. Importance refers "!Important" keyword , whichever styles has this rule , have more precedence

  3. Origin

    4. the Origin refers from where the styles are coming . is it browset defaults , user-defined styles At the end user-definred styles has more precedence

  4. order of Appererance

    5. if specificity , Origin , Importance same for an element , then the one which comes later has more precedence

Cascading Rules

  1. Id has the higher priority from class
  2. !Important has higher Priority from all
  3. inline has higher priority
  4. between External & intenal CSS , Both has same Priority but here Priority Works on order , Which one comes at last has the higher Priority
  5. in Internal CSS for same elements styles the last one will have higher priority
