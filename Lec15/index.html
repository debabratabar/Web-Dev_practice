Box -Sizing CSS Property

box-sizing property is allows us to include padding & border into an element's height & width
by default the height & width only applies to the content of the element , if we increase the border & padding then the height & width will increase
if we set to box-sizing value to "border-box" even if we increase the border & padding it will be in the defined height & width
